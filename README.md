# 딸깍 Resizer

클릭 한번으로 브라우저 크기를 저장해뒀던 값으로 변경해주는 익스텐션입니다. 옵션에서 크기를 지정할 수 있습니다. (기본값 1920x1080)

이 가난한 개발자는 $5이 없어 마켓에 등록을 하지 못합니다... 따라서 간단한 crx/xpi 설치 방법은 지원을 못합니다. 죄송합니다.

#### 설치 방법
 -  Chrome & Edge
 1. 압축 파일을 다운로드하고 압축을 해제합니다. [[링크]](https://github.com/darkpppet/DDalGGak-Resizer/releases/)
 2. 압축이 해제된 폴더를 적절한 위치에 옮겨 둡니다. (설치 후 폴더를 이동하거나 삭제하면 확장을 사용할 수 없습니다. 다시 설치해줘야 합니다.)
 - Chrome
   1. 브라우저를 켜서 **[<chrome://extensions/>]** 으로 이동합니다. (주소창)
   2. **[개발자 모드]** 를 켭니다.
   3. **[압축해제된 확장 프로그램 로드]** - 폴더 선택

 - Edge
   1. 브라우저를 켜서 **[<edge://extensions/>]** 으로 이동합니다. (주소창)
   2. 좌측 중단 **[개발자 모드]** 를 켭니다.
   3. **[압축 풀린 파일 로드]** - 폴더 선택

<br/>

 - FireFox
   1. xpi 파일을 다운로드합니다. [[링크]](https://github.com/darkpppet/DDalGGak-Resizer/releases/)
   2. FireFox [Nightly](https://www.firefox.com/channel/desktop/)/[Developer Edition](https://www.firefox.com/channel/desktop/)/[ESR](https://www.firefox.com/browsers/enterprise/) 중 하나를 설치합니다. (Developer Edition 추천)
   3. 브라우저를 켜서 **[<about:config>]** 로 이동합니다. (주소창)
   4. **[xpinstall.signatures.required]** 를 **[false]** 로 변경합니다.
   5. **[<about:addons>]** 로 이동합니다. (주소창)
   6. **[톱니 버튼]** - **[파일에서 부가 기능 설치]** - 파일 선택

<br/>

 - 다른 브라우저
   - 고려하지 않았습니다.
